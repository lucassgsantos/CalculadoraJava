# Calculadora Java

Calculadora com interface grafica feita em Java Swing, inspirada no design da calculadora do iOS.

## Funcionalidades

- Operacoes basicas: soma, subtracao, multiplicacao e divisao
- Raiz quadrada
- Porcentagem
- Inversao de sinal (+/-)
- Suporte a teclado (numeros, operadores, Enter, Backspace, Esc)
- Display com tamanho de fonte adaptavel
- Tratamento de divisao por zero

## Como executar

```
cd src
javac App.java Calculator.java
java App
```
